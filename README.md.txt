# NPS Mariposa Dashboard

Dashboard interactivo para análisis de Net Promoter Score (NPS).

## Características
- Análisis NPS por asesor
- Gestión de comentarios por categorías
- Exportación de datos
- Tendencias mensuales
- Autenticación con código

## Tecnologías
- Next.js 14
- React 18
- Tailwind CSS
- Supabase
- Recharts
